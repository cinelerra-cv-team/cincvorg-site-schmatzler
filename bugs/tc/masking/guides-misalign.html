Change format, and the mask outline and the mask don't line up

Use the mask-demo.tgz if you don't have any footage to play with.

The bug

If you create a mask, and change the output resolution, the mask outline and control points don't follow the actual mask anymore.

How to reproduce

Create a mask with a clear and obvious effect: E.g. a black square on a bright background. Or use the test project mask-demo.tgz, and use the gain slider in the patchbay to darken the "lower" video track (Video 1). You have to disable the "gang fader" to fade down the video tracks individually. Now you have very obvious mask effect, as shown in this screenshot.

Change the output format, e.g. to 1280x768 pixels. Enjoy the result. The mask outline has now very little to do with the actual mask.

Workaround?

We'd like to here about a workaround, if it exists.
