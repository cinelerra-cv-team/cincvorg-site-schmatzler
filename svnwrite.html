Getting SVN write access

Getting ssh write access

If you have several changes to Cinelerra's source code you want to contribute, we prefer that you check them in yourself. You need write access to the Subversion to do that.

To get write access, send an email to herman@skolelinux.no, include your desired user name and password hash (which may be created by for instance cryptit).

What do you get?

You get a shell account at skolelinux.no, also known as cvs.cinelerra.org. The home directory shares a couple of gigabytes with the other Cinelerra SVN users. It'll be almost full most of the time, so be nice!

You can put stuff on the Web by copying files to the public_html directory in your home directory. The server supports PHP and Perl cgi.
